#### Dependencies

#### Properties

> TimeAgoDirective

| Name     | Description                                  |
|----------|----------------------------------------------|
| [Date]   | Date                                         |
| [Locale] | Language code                                |
| [Live]   | Dynamically change. Default value: ``false`` |

> TimeAgoPipe

| Name    | Description                           |
|---------|---------------------------------------|
| args[0] | ``string`` type is **_Locale_** value |
| args[1] | ``boolean`` type is **_Live_** value  |
