#### Dependencies

- Lootstrap css library

#### Properties

| Name            | Description                              |
|-----------------|------------------------------------------|
| [Mode]          | Single, Multiple                         |
| [Items]         | Tree View items                          |
| [Sort]          | Sort the items. Default value: ``false`` |
| [SelectedItems] | EventEmitter, Returns selected items     |
